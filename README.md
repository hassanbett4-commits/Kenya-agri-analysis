# Kenya Agricultural Market Analysis (2019–2023)  Analysis of smallholder crop performance across 10 Kenyan 
counties using SQL, Excel, and Power BI.  ## Key Insights - Top 3 crops account for ~55% of total regional revenue - 
2022 drought caused measurable yield drops across most crops - Productivity gaps of 30–45% exist in several high
potential regions  ## Tools Used SQL (SQLite) | Microsoft Excel | Power BI  ## Files - `kenya_agri_analysis.sql` — Full 
SQL exploration (data quality, revenue, price trends, productivity gaps) - `kenya_agri_raw.csv` — Raw dataset (950 
records, 10 regions, 10 crops, 2019–2023)  ## Author Kipkoech Hassan Bett | Data Analyst | Nairobi, Kenya# Kenya-agri-analysis
